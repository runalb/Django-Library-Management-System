# Django - Inventory Management System

<b> First run this command to install all the libraries required to run the program </b>
- pip install -r requirements.txt

<b> To Run Project </b>
- python manage.py runserver

<b> Admin account: </b>
- username: admin
- Password: admin

<b> User account: </b>
- username: user
- Password: user@123
